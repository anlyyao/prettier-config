# prettier-config
my prettier config

### Install
```javascript
    yarn add -D @anlyyao/prettier-config
```

### Usage
#### 两种使用方式
#### 第一种，在 `package.json` 中

```javascript
  {
    "prettier": "@anlyyao/prettier-config"
  }
```

#### 第二种，在 `.prettierrc.js` 中
```javascript
	module.exports = require("@anlyyao/prettier-config");
```
